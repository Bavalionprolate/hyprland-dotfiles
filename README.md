# Моя рабочая оболочка для Hyprland

Данный репозиторий содержит мою рабочую оболочку для Hyprland с dotfiles, которая включает в себя различные компоненты и настройки для создания удобной и функциональной рабочей среды, основанной на GTK4, Ignis и Python. Она предоставляет полноценный опыт использования рабочего стола с настраиваемой панелью, доком, уведомлениями, центром управления и многим другим.

## Компоненты

Оболочка включает следующие компоненты рабочего стола:

- **Панель (Bar)**: Верхняя панель с системными индикаторами
- **Док (Dock)**: Панель запуска приложений и диспетчер задач
- **Центр управления (Control Center)**: Панель быстрых настроек с управлением громкостью, центром уведомлений, сетью и Bluetooth
- **Уведомления**: Всплывающие уведомления
- **Лаунчпад (Launchpad)**: Сетка приложений для запуска программ
- **Меню питания (Power Menu)**: Интерфейс для выключения, перезагрузки, блокировки, сна и выхода из системы
- **Панель обоев (Wallpaper Panel)**: Средство выбора обоев с предварительным просмотром
- **OSD**: Экранный дисплей для отображения изменений громкости

## Установка

### Предварительные требования

Убедитесь, что у вас установлен работающий Hyprland на Arch Linux перед установкой.

### Необходимые пакеты

```bash
paru -S ignis dart-sass ttf-jetbrains-mono-nerd slurp grim cmake meson cpio grimblast kitty xdg-desktop-portal-hyprland xdg-desktop-portal-gtk xdg-user-dirs uwsm telegram-desktop alsa-utils alsa-plugins alsa-lib playerctl imv wl-clipboard libnotify bibata-cursor-theme papirus-icon-theme hyprpicker-git gnome-bluetooth-3.0 blueman bluez-utils hyprlock
```

### Настройка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Bavalionprolate/hyprland-dotfiles.git
```

2. Перейдите в клонированный каталог:

```bash
cd hyprland-dotfiles
```

3. Скопируйте файлы конфигурации в каталог `.config`:

```bash
cp -r .config/* ~/.config/
```

4. Включите службу Bluetooth (если планируете использовать Bluetooth):

```bash
sudo systemctl enable bluetooth
```

> **Примечание:** Если у Вас по каким-то причинам не работает Bluetooth или вы не хотите его использовать, то просто удалите пакеты `gnome-bluetooth-3.0 blueman bluez-utils`

5. Запустите Hyprland с предустановленым конфигом из репозитория

## Конфигурация

> Данные настройки необходимы для многих виджетов и компонентов. Большенство из них настраиваются через интерфейс, но вы можете вручную изменить их в файл `settings.json` в папке `~/.config/ignis`.

## Настройка settings.json

Файл `~/.config/ignis/settings.json` содержит все пользовательские настройки системы. Вот полный обзор доступных опций:

### Настройки темы
```json
"theme": {
    "dark_mode": true,              // Включение темной темы
    "wallpaper": "/path/to/wallpaper.jpg",
    "monitor_wallpapers": [
        {
            "0": "/path/to/wallpaper.jpg"   // Монитор 0
        },
        {
            "1": "/path/to/wallpaper.jpg"   // Монитор 1
        }
    ],
    "wallpaper_dir": "/home/bava/Pictures/Wallpapers"   // Директория с обоями
    }
```

### Настройки приложений
```json
"apps": {
    "pinned_apps": [               // Закрепленные в доке приложения
        "org.gnome.Nautilus.desktop",
        "firefox.desktop",
        "kitty.desktop"
    ],
    "hidden_apps": [              // Скрытые из лаунчпада приложения
        "bssh.desktop",
        "bvnc.desktop",
        "avahi-discover.desktop"
    ]
}
```

### Настройки уведомлений
```json
"notifications": {
    "dnd": false,
    "notification_sound": true // пока что не реализован функционал
}
```

### Настройки дока
```json
"dock": {
    "auto_hide": true,           // Автоскрытие
    "icon_size": 44,            // Размер иконок
    "animation_speed": 200      // Скорость анимаций (мс)
}
```

### Настройки лаунчпада
```json
"launchpad": {
    "columns": 7,               // Количество колонок
    "rows": 3,                 // Количество рядов
    "grid_spacing_x": 160,     // Горизонтальный отступ сетки
    "grid_spacing_y": 140,     // Вертикальный отступ сетки
    "margin": 140,             // Отступ от краев
    "app_size": 160,           // Размер элемента приложения
    "icon_size": 48,           // Размер иконки
    "invert_scroll": true,     // Инверсия прокрутки
    "page_scale_x": 0.8,       // Масштаб страницы по горизонтали
    "page_scale_y": 1.0        // Масштаб страницы по вертикали
}
```

## Виджеты и компоненты

### Панель (Bar)

Верхняя панель рабочего стола предоставляет быстрый доступ к основным функциям:
- **Системное меню**: Показывает текущее время, дату
- **Трей**: Отображает системные индикаторы и значки приложений
- **Индикаторы**:
  - Аудио устройства и управление громкостью
  - Раскладка клавиатуры
  - Переключатель для отображения центра управления
- **Кнопка перемещения**: Перемещает на новое рабочее пространство
- **Виджет рабочий пространств**:
  - Перемещение между открытыми рабочими пространствами
  - Отображает текущее рабочее пространство
  - Отображает используемые пространства

### Док (Dock)

Настраиваемая панель приложений с расширенными возможностями:
- **Закрепленные приложения**:
  - Настройка через settings.json или контекстное меню по приложению
  - Drag-and-drop для изменения порядка
- **Активные приложения**:
  - Индикация запущенных приложений
  - Группировка окон
- **Дополнительные функции**:
  - Автоскрытие (настраивается в settings.json или в контекстном меню кнопки лаунчпада)
  - Настраиваемый размер иконок (настраивается в settings.json)

### Центр управления (Control Center)

Комплексный центр управления системой:
- **Аудио управление**:
  - Отдельные слайдеры для системы и приложений
  - Переключение устройств ввода/вывода
  - Быстрое отключение звука
- **Bluetooth**:
  - Быстрое включение/выключение
  - Управление сопряженными устройствами
  - Подключение новых устройств
- **Сеть**:
  - Статус подключения
  - Список доступных сетей
  - Быстрое подключение/отключение
- **Тема**:
  - Переключение светлой/темной темы
- **Уведомления**:
  - История уведомлений
  - Очистка всех уведомлений
  - Режим "Не беспокоить"

### Система уведомлений

Современная система уведомлений с широкими возможностями:
- **Всплывающие уведомления**
- **Взаимодействие**:
  - Кнопки действий
  - Закрытие
- **Настройки**:
  - Режим "Не беспокоить"

### Лаунчпад (Launchpad)

Многофункциональный запускатель приложений:
- **Сетка приложений**:
  - Настраиваемое количество колонок и строк
  - Пагинация с плавной прокруткой
  - Адаптивные размеры иконок
- **Поиск**:
  - Мгновенный поиск по названию и описанию
  - Web-поиск для неустановленных приложений
- **Управление**:
  - Скрытие приложений
  - Закрепление в доке
  - Настройка через контекстное меню

### Панель обоев (Wallpaper Panel)

Продвинутый менеджер обоев рабочего стола:
- **Галерея**:
  - Предпросмотр всех обоев
  - Поддержка различных форматов
  - Выбор разных обоев на мониторы
- **Управление**:
  - Выбор пользовательской директории
  - Мгновенное применение
  - Запоминание истории

### Меню питания (Power Menu)

Стильное меню управления питанием:
- **Действия**:
  - Выключение системы
  - Перезагрузка
  - Спящий режим
  - Блокировка (hyprlock)
  - Выход из сессии

### OSD (Экранный дисплей)

Показывает временные наложения для изменения громкости

## Сочетания клавиш

В Hyprland настроены следующие сочетания клавиш:

### Основные команды

- `SUPER + RETURN` - Открыть терминал (kitty)
- `SUPER + Q` - Закрыть активное окно
- `SUPER + M` - Выйти из Hyprland
- `ALT + E` - Открыть файловый менеджер (nautilus)
- `SUPER + V` - Переключить плавающий режим окна
- `ALT + S` - Открыть лаунчпад (меню приложений)
- `SUPER + P` - Псевдо-тайлинг
- `SUPER + J` - Переключить разделение
- `SUPER + F` - Полноэкранный режим (с сохранением панелей)
- `SUPER + SHIFT + F` - Полный полноэкранный режим
- `ALT + Tab` - Переключение между окнами
- `SUPER + SHIFT + E` - Сделать все окна плавающими
- `SUPER + E` - Переключить плавающий режим активного окна

### Утилиты

- `SUPER + SHIFT + S` - Сделать скриншот выбранной области
- `PRINT` - Сделать скриншот текущего экрана
- `ALT + P` - Закрепить окно поверх всех
- `CTRL + P` - Выбрать цвет с экрана (hyprpicker)
- `SUPER + SHIFT + W` - Открыть панель обоев

### Управление мониторами и рабочими пространствами

- `SUPER + SHIFT + CTRL + стрелка влево/вправо` - Переместить окно на другой монитор
- `SHIFT + CTRL + стрелка влево/вправо` - Переместить текущее рабочее пространство на другой монитор
- `SUPER + Z` - Переключить специальное рабочее пространство (scratchpad)
- `SUPER + SHIFT + Z` - Переместить окно в специальное рабочее пространство

### Управление фокусом и перемещение окон

- `SUPER + стрелки` - Перемещение фокуса между окнами
- `SUPER + SHIFT + стрелки` - Перемещение активного окна

### Переключение рабочих пространств

- `SUPER + [1-9,0]` - Переключение на рабочее пространство 1-10
- `SUPER + SHIFT + [1-9,0]` - Переместить активное окно на рабочее пространство 1-10
- `SUPER + колесо мыши вверх/вниз` - Прокрутка между рабочими пространствами
- `SUPER + Page Up/Down` - Переключение на следующее/предыдущее рабочее пространство

### Изменение размера и перемещение окон

- `SUPER + ЛКМ` - Перемещение окна
- `SUPER + ПКМ` - Изменение размера окна
- `SUPER + ALT + стрелки` - Изменение размера активного окна

### Мультимедийные клавиши

- `XF86AudioRaiseVolume` - Увеличить громкость и показать OSD
- `XF86AudioLowerVolume` - Уменьшить громкость и показать OSD
- `XF86AudioMute` - Включить/выключить звук и показать OSD
- `XF86AudioMicMute` - Включить/выключить микрофон
- `XF86AudioNext` - Следующий трек
- `XF86AudioPause/Play` - Воспроизведение/пауза
- `XF86AudioPrev` - Предыдущий трек
